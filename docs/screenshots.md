- Título y subtítulo completos

## Herramientas recomendadas

- **Chrome DevTools** → Device Toolbar → Desktop (1440px)
- **Extensión:** GoFullPage o FireShot para captura de página completa
- **Formato:** PNG, sin compresión
- **Ubicación:** Guardar en `public/screenshots/` o subir a la carpeta de assets del proyecto
